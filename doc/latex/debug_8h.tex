{\footnotesize\ttfamily enum \mbox{\hyperlink{debug_8h_a44cbbdcfb32a8f38245ff377ef1c7b04}{debug\+Levels}}}

Initialize the variable {\ttfamily debug\+Level} depending on the value of {\ttfamily argv\mbox{[}1\mbox{]}}. Normally called from {\ttfamily main} with the program arguments. If {\ttfamily argv\mbox{[}1\mbox{]}} is or begins with {\ttfamily -\/debug}, the value of {\ttfamily debug\+Level} is set and {\ttfamily argc, argv} are modified appropriately. An entry of {\ttfamily -\/debug5} sets the level to 5. If the function is not called, the user is responsible for setting {\ttfamily debug\+Level} in other code.
